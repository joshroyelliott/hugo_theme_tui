Every titled container on this site is partials/panel.html called with a different variant. Pass a dict with title, number, variant, and content:

{{ partial "panel.html" (dict
  "title"   "Posts"
  "variant" "list"
  "items"   .Pages
) }}

List + Preview

A list panel paired with a preview panel. Clicking an item in the list swaps the preview content without a page navigation.

[1] Items
  • First itemMarch 1, 2025

[2] First item

This is the preview for the first item. Click a different entry in the list panel to swap the content here.

On real list pages (like the blog index), list-preview.js fetches index.json once and swaps from memory. This demo uses inline content instead.

Second item selected. The preview title updates and the list panel counter tracks which item is active.

The scroll indicator on the right edge of each panel updates as content changes height.

Third item. On pages without JavaScript the list items are plain links that navigate to the full page — the preview is a progressive enhancement.

On real section pages (like the blog index) list-preview.js fetches index.json once and renders from memory. Without JavaScript the list items are plain links to the full pages.

Tabs

Dual-mode: tab switchers in the border when narrow, side-by-side columns when wide. Resize the window to see both states.

Tabs
Narrow-Wider-Keyboard
(resize window)

Below the width threshold, tab names in the border act as switchers.

Once wide enough, all tabs render side-by-side as columns.

Focus a tab and use ArrowLeft/ArrowRight to navigate.

Input

A prompt-style wrapper. The border and title switch to --base0C on focus and the textarea auto-grows as you type.
