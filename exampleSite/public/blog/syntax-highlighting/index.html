Chroma emits CSS classes (the theme sets markup.highlight.noClasses = false), and _chroma.scss maps those classes to var(--base0E), var(--base0B), var(--base09), and friends. Swap the site’s data/colorscheme.yaml and every token colour updates in lock-step with the rest of the page.

 Go

package main

import "fmt"

// greet prints a welcome message.
func greet(name string) {
    fmt.Printf("Hello, %s!\n", name)
}

func main() {
    greet("world")
}

 Python

def fib(n):
    """Return the n-th Fibonacci number."""
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a

 Rust

use std::collections::HashMap;

fn count_words(text: &str) -> HashMap<&str, usize> {
    let mut counts = HashMap::new();
    for word in text.split_whitespace() {
        *counts.entry(word).or_insert(0) += 1;
    }
    counts
}

Inline & prose

You can also drop inline code into a paragraph — it uses var(--base0A) on the var(--base01) surface, so it stands out without clashing. External links like Hugo’s docs open in a new tab via the render-link.html hook; internal links stay in-page.
