The footer sits at the bottom of every page via partials/body/footer.html. Copyright on the left, social links on the right, separated by |.

With icons

Each link can carry a glyph from data/icons.yaml. Below 768px the text hides, leaving icon-only links.

Config

Links come from params.social in hugo.toml:

[params]
  copyright = "&copy; 2025 tui demo"
  [[params.social]]
    name = "GitHub"
    url = "https://github.com/user"
    icon = "github"
  [[params.social]]
    name = "RSS"
    url = "/index.xml"
    icon = "rss"

External URLs open in a new tab; internal URLs (starting with /) stay in-page. Omit copyright to default to © {year} {site.Title}.
